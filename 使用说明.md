# AI 会话管家 - 使用说明

## 快速启动

### 方式 1: 双击批处理文件（推荐）

直接双击以下任一文件启动：

- **`启动应用.bat`** - 显示控制台日志，方便调试
- **`SessionShelf.bat`** - 无控制台窗口，后台静默运行

### 方式 2: 命令行启动

```bash
cd D:\SoftwareCoding\workspace\LGX\ai\SessionShelf
export JAVA_HOME="C:\Users\LiGuoXian\.jdks\ms-17.0.16"
mvn javafx:run
```

## 文件说明

## 创建 EXE 安装程序

### 前置条件

需要安装 **WiX Toolset**：

1. 访问 https://wixtoolset.org/
2. 下载 WiX Toolset v3.11 或更高版本
3. 安装时勾选"Add to PATH"选项

### 打包步骤

1. 确保已安装 WiX Toolset
2. 双击运行 `打包成EXE.bat`
3. 等待打包完成
4. EXE 文件将生成在 `target\dist\SessionShelf-1.0.0.exe`

## 创建桌面快捷方式

1. 右键 `SessionShelf.bat` -> 发送到 -> 桌面快捷方式
2. 右键快捷方式 -> 属性 -> 更换图标（可选）

## 故障排查

### 问题: 启动时提示"找不到 Java"

**解决**: 确保 JAVA_HOME 指向正确的 JDK 17 路径

### 问题: 界面显示异常

**解决**: 检查显卡驱动是否支持 JavaFX 硬件加速

### 问题: 无法读取 Claude Code 数据

**解决**: 确认 Claude Code 数据目录存在：
- Windows: `%USERPROFILE%\.claude\projects\`
- macOS: `~/.claude/projects/`
